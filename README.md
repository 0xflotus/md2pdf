## How to use md2pdf?
1. Click button choose `.md` file
2. Edit in editor (left panel).
3. Click **Transform**!
4. **Chrome recommended**

## Tips
- `Resize` the layout what you want.
- After click `Transfom` button, inverse the checkbox of **'Headers and Footers'**. 
- **反選頁首與頁尾**.
